
Для того, чтобы сдать, должны быть выполнены требования:
+• Данные должны считываться в массивы
+• Код должен быть разделён на функции вменяемым образом (не main с вызовом одной функции, которая всё внутри делает)
+• Нужно проверять, что все файлы открываются, и принимать соответствующие меры, если не открылись (сообщить об ошибке и завершить программу)
+• Не должно быть проблем со стилем (разные отступы, скобки по-разному расставлены, переменные с непонятными именами и тд)
+• Не должно быть избегаемого копипаста, то есть обнулить 3 массива тремя memsetами можно, но переписывать функцию, чтобы считать то же самое на других данных, не надо

Если все условия выполнены, можно получить оценку 4 и дальше показывать доп фичи без дедлайна
Если не выполнены, то есть 2 недели (до конца октября), чтобы дописать код

Доп фичи:
-• учитывание tolerance +1
+• остальные проверки (выделение памяти, выходы за границы массива и тд) + 0.5
+• github +1.5
+• динамические массивы + 0.5
+• расчёт погрешностей/много вычислений +1.5
